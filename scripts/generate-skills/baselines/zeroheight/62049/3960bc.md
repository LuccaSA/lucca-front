# Angular

## Chip Angular `v19.2`

Mots-clés : étiquette, filtres

Component selector : `lu-chip`

```ts

```

### Ellipse

Il est nécessaire d'ajouter une balise `span` avec l'[utilitaire](https://prisme.lucca.io/94310e217/p/21a286-utilitaires/t/64af3b) `u-ellipsis` ainsi qu'une largeur autour du label afin d'obtenir une ellipse.
