# Content

## Imaginons une personne incarnant les solutions Lucca :

* Cette personne, sincère, **a vraiment envie d'aider l'utilisateur**. 
* Elle **parle de façon agréable et correcte**, sans trop en faire. 
* Elle **encourage ou réoriente** au besoin dans l'exécution des tâches. 
* Elle **s'autorise même, de temps en temps, un clin d'œil** amical ou un mot d'esprit. 
* En somme, elle **fournit un excellent niveau de service, tout sauf gris, moche ou triste.** 

	À nous de retranscrire fidèlement sa voix à travers les solutions Lucca.

---

## Restons à la fois courtois, pro et simple

* **Adressons-nous directement à l'utilisateur** en évitant les pronoms neutres.
* **Vouvoyons** l'utilisateur.

| ✅ | ❌ |
| --- | --- |
| Votre demande de congés a été envoyée. | Ta demande de congés a été envoyée.  |

* **Écartons les capitales**, qui donnent l'impression de CRIER SUR l'UTILISATEUR.

| ✅ | ❌ |
| --- | --- |
| Envoyer | ENVOYER |

* **Quand nous nommons, désignons une personne, c'est par son** ***Prénom Nom*****.** 

Jamais par "*M./Mme Nom*" ni par "*Nom Prénom*".

<callout background="3" fullWidth="true">

* **Si on s’adresse directement à l’utilisateur, on l’appelle directement par son "*****Prénom"*****.**

| ✅ | ❌ |
| --- | --- |
| Marie Bragoulet a validé votre demande. | Votre demande a été validée par Madame Bragoulet. |
| Bonjour Marie. | Bonjour Marie Bragoulet. |

## Favorisons la pédagogie

* **Exprimons-nous toujours sous un angle positif**, en évitant au maximum les négations.

| ✅ | ❌ |
| --- | --- |
| Votre dépense sera sauvegardée quand la connexion internet sera rétablie. | Vous avez perdu votre connexion, la dépense n’a pas pu être sauvegardée. |
| Vous avez des demandes de congés à approuver. | Vous n’avez pas approuvé toutes les demandes de congé de vos collaborateurs. |

* **Expliquons toujours un acronyme**, soit dans le texte, soit via un **[tooltip](https://prisme.lucca.io/94310e217/p/9285d2)**.

| Rule | Image | Caption | Description |
| :--- | :--- | :--- | :--- |
| Do |  |   |   |
| Don't |  |   |   |
| Do |  |   |   |
| Don't |  |   |   |

* Efforçons-nous toujours de **proposer une solution**.

| ✅ | ❌ |
| --- | --- |
| Pour votre sécurité, votre mot de passe doit contenir entre 8 et 13 caractères ainsi qu'un caractère spécial. | Votre mot de passe n’est pas suffisamment sécurisé. |

* **Ne mettons pas en faute** l'utilisateur.

| ✅ | ❌ |
| --- | --- |
| Nous ne reconnaissons pas l’adresse email que vous avez indiquée. | Vous avez fourni un email incorrect. |

* **Ne donnons pas d'ordre**.

Employer "Merci de", c'est donner un ordre déguisé. Idem pour "Veuillez", qui alourdit aussi les phrases. 

Gardons un [ton conversationnel](https://prisme.lucca.io/94310e217/p/701bd6) : préférons inciter à l'action avec un verbe à l'impératif, ou un "Pensez à".

| ✅ | ❌ |
| --- | --- |
| Renseignez le champ prix. | Le champ “Prix” est obligatoire. |
| Pensez à déclarer vos temps. | Merci de déclarer vos temps. Veuillez déclarer vos temps. |
