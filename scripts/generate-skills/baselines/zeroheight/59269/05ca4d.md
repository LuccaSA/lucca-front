# Angular

## Table of content Angular `v21.0`

```ts
import { TableOfContentComponent, TableOfContentLinkDirective } from '@lucca-front/ng/table-of-content';
```

[Basic](https://lucca-front.lucca.io/master/storybook/iframe.html?id=documentation-navigation-tableofcontent-angular-basic--basic)

# HTML

## Table of content HTML

```css
@forward '@lucca-front/scss/src/components/tableOfContent';
```

[Basic](https://lucca-front.lucca.io/master/storybook/iframe.html?id=documentation-navigation-tableofcontent-html-css-basic--basic)
