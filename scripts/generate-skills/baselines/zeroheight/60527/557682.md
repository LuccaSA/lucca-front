## Angular

Directive selector : `luDropdown`

```ts
import { LuDropdownTriggerDirective, DropdownMenuComponent, DropdownItemComponent, DropdownActionComponent, DropdownGroupComponent } from '@lucca-front/ng/dropdown';
```

[Directive](https://lucca-front.lucca.io/master/storybook/iframe.html?id=documentation-overlays-dropdown-angular-directive--directive)

Depuis la version [20.3](https://github.com/LuccaSA/lucca-front/releases/tag/v20.3.0), l'utilisation de `<lu-dropdown>` a été dépréciée au profit d'un template pour des questions de performances.
