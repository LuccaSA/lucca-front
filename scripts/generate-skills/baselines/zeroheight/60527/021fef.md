# HTML

## Title and text HTML

```css
@forward '@lucca-front/scss/src/components/title';
```

[Basic](https://lucca-front.lucca.io/storybook/iframe.html?id=documentation-texts-text-basic--basic)

<callout background="1">

**v19.2**

Les marges et paddings sont supprimés par défaut pour éviter l'usage massif d'utilitaires de spacings.

</callout>

Il est possible de conserver ces spacings temporairement via un flag suivant :

```css
@use '@lucca-front/scss/src/commons/config' with (
  $deprecatedSpacings: true,
);
```

## Règles d’usage

### Accessibilité 

* Pour faciliter le travail de construction de plans de documents de nos pages, la sémantique des titres et leurs styles graphiques peuvent être dissociés avec ces classes utilitaires : `.pr-u-h1, .pr-u-h2…`.
* Des *mixins* `@include hx, @include h1, @include h2…` permettent également de pouvoir associer des styles graphiques différents en fonction du *responsive* ; le *mixin* `@include hx` regroupant les styles communs à tout les niveaux de titres.

<callout background="3" fullWidth="true">

Pour tous les Titles, [ne mettons pas de point final.](https://prisme.lucca.io/94310e217/p/63277d-ponctuation-typographie/t/751eb7)

</callout>

# iOS

## Title and text iOS

```swift
import LuccaSDK
```

Sur tous les textes des maquettes Figma est spécifié un style de texte définit dans [Prisme](https://prisme.lucca.io/94310e217/p/73bd2f-typographie/t/91a124). Ce style décrit une police, une taille, une graisse et une couleur. Afin de simplifier la création des vues `Text("...")`, le SDK propose 3 `ViewModifier` qui appliquent la mise en forme appropriée.

```swift
// Applique la font Lucca, la taille de police, la graisse et la couleur.
.luccaTextStyle(prisme: LuccaTextStyleConstants)

// Applique seulement la font Lucca, la taille de police et la graisse.
.luccaTextFont(prisme: LuccaTextStyleConstants)

// Applique seulement la couleur.
.luccaTextColor(prisme: LuccaTextStyleConstants)

```

### Exemple

```swift
import LuccaSDK

// Affiche le texte "Hello world" avec le style Prisme "Body 1"
// - font & graisse : SourceSansPro-Regular
// - font-size : 16pts
// - couleur: Neutral800
Text("Hello world")
   .luccaTextStyle(prisme: .body1)
```

### Equivalences Prisme vs. LuccaTextStyleConstants

<callout background="1">

Suivons les [recommandations d'Apple](https://developer.apple.com/design/human-interface-guidelines/typography#Specifications) en matière de lisibilité : appuyons-nous uniquement sur les styles de texte Prisme dispo pour les maquettes destinées à iOS.

</callout>
