---
title: Sortable list
description: Une Sortable list désigne une liste d'éléments pouvant être réorganisée par l'utilisateur en les faisant glisser et en les déposant à l'endroit souhaité.
---

# Angular

## Sortable list Angular `v21.0`

Le drag and drop est porté par Angular CDK [drag-drop](https://angular.dev/guide/drag-drop).

Component selector : `lu-sortable-list`

```ts
import { SortableListComponent, SortableListItemComponent } from '@lucca-front/ng/sortable-list';
```

[Basic](https://lucca-front.lucca.io/storybook/iframe.html?id=documentation-listings-sortable-list-angular-draggable--basic)
