---

# Design

## Règles d'utilisation

### Fil d’ariane ou bouton de retour ?

| Rule | Image | Caption | Description |
| :--- | :--- | :--- | :--- |
| Do |  |   | Utilisons le fil d’ariane (breadcrumb) s’il existe plusieurs niveaux de profondeurs. Cela permet à l’utilisateur de se situer dans l’application et de remonter de plusieurs niveaux si besoin. |
| Do |  |   | Utilisons le bouton de retour s’il n’y a qu’un seul niveau de profondeur. |
| Don't |  |   | Ne cumulons pas le fil d’ariane et le bouton de retour. Utilisons celui qui correspond au contexte et au besoin de l’utilisateur. |
