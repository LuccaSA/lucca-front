Forms

Form structure

To create a complete form layout, place each form element in a form-group container. See template example

<form class="form"> <div class="form-group"> Form element. e.g. input </div> <div class="form-group"> Form element. e.g. checkbox </div> <div class="form-actions"> Action buttons </div> </form> Tip: You can use form elements like input, checkbox or radio outside a form structure: it will reset margins. It can be useful for a custom use, for exemple, in tables.

Input

<div class="input"> <input class="input-field" type="text"> <label class="input-label">Input label</label> </div> Tip: Use <div class="input mod-block"> for a 100% width input.
Tip: Use <div class="input is-filled"> to let the placeholder on the upper state.

Checkboxes

<div class="checkbox"> <input class="checkbox-input" type="checkbox" id="checkboxID"> <label class="checkbox-label" for="checkboxID">Checkbox</label> </div> Tip: To disable a checkbox, add a disabled attribute to the checkbox input.

Incomplete checkbox

For an incomplete checkbox, add class is-incomplete to input checkbox-input.

Radio

<div class="radio"> <input class="radio-input" type="radio" name="radioListID" id="radioID" checked> <label class="radio-label" for="radioID">Radio</label> </div> Tip: To disable a radio, add a disabled attribute to the radio input.
