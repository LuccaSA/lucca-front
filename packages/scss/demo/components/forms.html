Forms

Form structure

To create a complete form layout, place each form element in a form-group container. See template example

<form class="form"> <div class="form-group"> Form element. e.g. input </div> <div class="form-group"> Form element. e.g. checkbox </div> <div class="form-actions"> Action buttons </div> </form> Tip: You can use form elements like input, checkbox or radio outside a form structure: it will reset margins. It can be useful for a custom use, for exemple, in tables.
Tip: You can use a form-group-label over every form elements.
