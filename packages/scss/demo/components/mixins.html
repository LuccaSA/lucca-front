Mixins & Functions

Theming Functions

These are functions you can use to access variables from the theme. They are available by adding @import "~@lucca-front/scss/src/theming.overridable" in your file. They return variables from maps. They use the dot notation parent.child.[...].

Color/Palettes

_color("palette_name") // returns the palette.color value _color("palette_name", "key") // return the palette.key value

Component

_component("componentname.key.[...]")

Theme

This is the generic way to access our maps

_theme("mapname.key.[...]")

Get

If you need to access value in your own map, you can use the _get() function

_get($mapname, "key.[...]")

Mixins

These mixins are available by adding @import "~@lucca-front/scss/src/mixins"; in your file

Elevated

You can add shadow to your own component by using elevated(1-5) mixin

@include elevate(1);

Loader

You can use this mixins to turn your element into a spinner

@include loading($size);

Media queries

Using the breakpoint values in the breakpoints map, you can easily handle your media queries

@include media_smaller_than("breakpoint_key"){} // <==> @media (max-width: breakpoint_value - 1px) {} @include media_larger_than("breakpoint_key"){} //<==> @media (min-width: breakpoint_value) {} @include media_between("breakpoint1_key", "breakpoint2_key"){} //<==> @media (min-width: breakpoint1_value) and (max-width: breakpoint2_value - 1px) {}
