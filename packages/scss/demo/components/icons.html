Icons

Icons

heart <i class="lucca-icon">heart</i>

Just replace the text between <i></i> by the keyword in this list to get your icon.

{{icon-name}} {{icon-name}}
Tip: If you want to avoid ligatures, you can use the keyword as class in camelCase with the prefix icon-. e.g. <i class="lucca-icon icon-chevronSouth"></i>

Icons size

You can add a mod to change your icon size:

heart <i class="lucca-icon size-smaller">heart</i>
heart <i class="lucca-icon size-small">heart</i>
heart <i class="lucca-icon">heart</i>
heart <i class="lucca-icon size-big">heart</i>
heart <i class="lucca-icon size-bigger">heart</i>
heart <i class="lucca-icon size-biggest">heart</i>
Tip: You can manually set the icon size by giving it a font-size value.

Icons colors

You can use color utilities to change your icon color:

heart heart heart heart heart heart heart

Action icons

You can use icons as action:

<button class="actionIcon"> <i class="lucca-icon">hearth</i> </button>

Disabled

<button class="actionIcon" disabled> <i class="lucca-icon">heart</i> </button> Tip: actionIcon can also be disabled with a class class="actionIcon is-disabled", but you will not be able to use a tooltip on it.

Loading

<button class="actionIcon is-loading"> <i class="lucca-icon">trash</i> </button>

Inverted

<button class="actionIcon mod-invert"> <i class="lucca-icon">trash</i> </button>
