Icons size

You can add a mod to change your icon size:

heart <i class="lucca-icon size-smaller">heart</i>
heart <i class="lucca-icon size-small">heart</i>
heart <i class="lucca-icon">heart</i>
heart <i class="lucca-icon size-big">heart</i>
heart <i class="lucca-icon size-bigger">heart</i>
heart <i class="lucca-icon size-biggest">heart</i>
Tip: You can manually set the icon size by giving it a font-size value.

Icons colors

Action buttons

You can use icons as action buttons:

<button class="actionIcon"> <i class="lucca-icon">hearth</i> </button>

Disabled

<button class="actionIcon" disabled> <i class="lucca-icon">heart</i> </button>
