Icons

Icons

heart <span aria-hidden="true" class="lucca-icon">heart</span>

Just replace the text between <span aria-hidden="true"></span> by the keyword in this list to get your icon.

{{icon-name}} {{icon-name}}
Tip: If you want to avoid ligatures, you can use the keyword as class in camelCase with the prefix icon-. e.g. <span aria-hidden="true" class="lucca-icon icon-chevronSouth"></span>

Icons size

You can add a mod to change your icon size:

heart <span aria-hidden="true" class="lucca-icon size-smaller">heart</span>
heart <span aria-hidden="true" class="lucca-icon size-small">heart</span>
heart <span aria-hidden="true" class="lucca-icon">heart</span>
heart <span aria-hidden="true" class="lucca-icon size-big">heart</span>
heart <span aria-hidden="true" class="lucca-icon size-bigger">heart</span>
heart <span aria-hidden="true" class="lucca-icon size-biggest">heart</span>
Tip: You can manually set the icon size by giving it a font-size value.

Icons colors

You can use color utilities to change your icon color:

heart heart heart heart heart heart heart

Action icons

You can use icons as action:

<button class="actionIcon"> <span aria-hidden="true" class="lucca-icon">hearth</span><span class="u-mask">Like</span> </button>

Disabled

<button class="actionIcon" disabled> <span aria-hidden="true" class="lucca-icon">heart</span><span class="u-mask">Like</span> </button> Tip: actionIcon can also be disabled with a class class="actionIcon is-disabled", but you will not be able to use a tooltip on it.

Loading

<button class="actionIcon is-loading"> <span aria-hidden="true" class="lucca-icon is-loading">trash</span><span class="u-mask">Delete</span> </button>

Inverted

<button class="actionIcon mod-invert"> <span aria-hidden="true" class="lucca-icon">trash</span><span class="u-mask">Delete</span> </button>

Palette

<button class="actionIcon palette-error"> <span aria-hidden="true" class="lucca-icon">arrow_full_east</span><span class="u-mask">Next</span> </button>
