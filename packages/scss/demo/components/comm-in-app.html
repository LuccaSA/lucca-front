Communication in App

Toasts Comm In App Work in progress

Toasts Comm In App allow you to show information about new module, on a corner (bottom left) of your app.

warningWarning: You have to manually create and kill your toasts for now, they will be added in the NG package in the future.

You have to use the same code as toast, BUT you need to add the mod-commInApp class to the div.toasts

<div class="toasts mod-commInApp"> <div class="toasts-item"> <button class="toasts-item-kill"></button> ... </div> <div class="toasts-item"> <button class="toasts-item-kill"></button> ... </div> </div>

If you want to use the same template for your comm In app, we recommand you to copy this code into your innerhtml toast item

<img src="../images/commInApp.svg"> <div class="toasts-content"> <div class="toasts-content-title"><strong>Le module fait peau neuve !</strong></div> <p>Carentibus et vates arduos quam gloriae honeste ob primo.</p> <button class="button mod-outline mod-white mod-smaller">Essayez maintenant</button> <button class="button mod-link mod-invert mod-smaller">Plus tard</button> </div>
