Text format

Align

You can align your text by adding a class on any element:

Left align

Center align

Right align

u-textLeft u-textCenter u-textRight

Float

You can also align block elements by using a float:

Float left

Float right

u-left u-right

Tip: If needed, clear it by using <div class="u-clear"></div>.

Positioning

You can center vertically an inline element:

u-middle

Sizes

You can also change the size of your text:

smaller small standard big bigger biggest

u-textSmaller u-textSmall u-textStandard u-textBig u-textBigger u-textBiggest

Colors

You can also change the color of your text:

Default Light Placeholder Primary Secondary Success Warning Error

u-textDefault u-textLight u-textPlaceholder u-textPrimary u-textSecondary u-textSuccess u-textWarning u-textError

Whitespace

You can force a white-space: nowrap property. It can be useful to force a one line string in a table cell.

u-nowrap

Ellipsis

You can use ellipsis on a one line text to hide its extra part. You'll may need to define a width.

Cotton candy cupcake sweet roll. Oat cake jelly chocolate carrot cake cotton candy macaroon croissant.
u-ellipsis
