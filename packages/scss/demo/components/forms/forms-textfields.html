Inline validation

You can add a is-loading and a is-valid or is-invalid class on <div class="textfield"> for your inline validation.

Suffix

You can use a class mod-withSuffix to support a suffix on your input.

€
€
€
<div class="textfield mod-withSuffix"> <input class="textfield-input" id="textfieldID" type="text"> <label class="textfield-label" for="textfieldID">Textfield label</label> <div class="textfield-suffix">€</div> </div> Tip: If you are using a classic textfield without label, you'll need to add a mod-noLabel class.

Inline

Inline textfields works well with mod-material, but you can also use it with standard style.

<div class="textfield mod-inline"> <input class="textfield-input" type="text"> <label class="textfield-label">...</label> </div>
Tip: You can change the input lenght with mod-longest,mod-longer,mod-long, mod-short,mod-shorter or mod-shortest.
Tip: You can use inline input inside standard text:

Search

<div class="textfield mod-search"> <input class="textfield-input" id="textfieldID" type="text" placeholder="e.g. Clark Kent"> <label class="textfield-label" for="textfieldID">Search</label> </div> Tip: You can change the input style using mod-material & mod-compact

Required

<div class="textfield is-required"> <input class="textfield-input" id="textfieldID" type="text"> <label class="textfield-label" for="textfieldID">...</label> </div>

Palette

You can use your palette on inputs.

<div class="textfield palette-primary"> <input class="textfield-input" id="textfieldID" type="text"> <label class="textfield-label for="textfieldID">...</label> </div> Tip: It will works with all palette's colors
Tip: It also works with input using mod-material & mod-compact

Inverted

<div class="textfield mod-invert"> <input class="textfield-input" id="textfieldID" type="text"> <label class="textfield-label for="textfieldID>...</label> </div>

Multilines

<div class="textfield mod-multiline"> <textarea class="textfield-input" id="textfieldID"></textarea> <label class="textfield-label" for="textfieldID">Textarea label</label> </div> Tip: Use <div class="textfield mod-block"> for a 100% width textarea.
Tip: Use <div class="textfield-input is-filled"> to let the label on the upper state with mod-material.
