Input

<div class="input"> <input class="input-field" type="text"> <label class="input-label">...</label> </div> Tip: Use <div class="input mod-block"> for a 100% width input.
Tip: Use <div class="input-field is-filled"> to let the label on the upper state.
Tip: You can use placeholder="..." on input-field.

Helper / Error

Few tips about the input
<div class="input"> <input class="input-field" type="text"> <label class="input-label">...</label> <div class="input-helper">...</div> </div>
Oops, there is an error!
<div class="input"> <input class="input-field is-error" type="text"> <label class="input-label">...</label> <div class="input-error">...</div> </div>

Inline

<div class="input mod-inline"> <input class="input-field" type="text"> <label class="input-label">...</label> </div>
Tip: You can change the input lenght with mod-longest,mod-longer,mod-long, mod-short,mod-shorter or mod-shortest.
Tip: You can use inline input inside standard text:

<div class="input mod-search"> <input class="input-field" type="text" placeholder="e.g. Clark Kent"> <label class="input-label">Search</label> </div>

Palette

You can use your palette on inputs.

<div class="input palette-primary"> <input class="input-field" type="text"> <label class="input-label">...</label> </div> Tip: It will works with all palette's colors
