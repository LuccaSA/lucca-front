Checkboxes

<div class="checkbox"> <input class="checkbox-input" type="checkbox" id="checkboxID"> <label class="checkbox-label" for="checkboxID">Checkbox</label> </div> Tip: To disable a checkbox, add a disabled attribute to the checkbox input.

Incomplete checkbox

For an incomplete checkbox, add class is-incomplete to input checkbox-input.

Palette

<div class="checkbox palette-primary"> <input class="checkbox-input" type="checkbox" id="checkboxID"> <label class="checkbox-label" for="checkboxID">...</label> </div> Tip: It will works with all palette's colors
