Auto width

You can remove the columns numbers to devide the grid uniformly:

grid-md
grid-md
grid-md
grid-md
grid-md
<div class="grid"> <div class="grid-md">...</div> <div class="grid-md">...</div> ... </div> Tip: It will also work with -xs, -sm, -md, -lg

Horizontal alignment

You can align columns horizontaly inside the grid:

<div class="grid mod-start-xs"> ... <div class="grid mod-center-xs"> ... <div class="grid mod-end-xs"> ... Tip: It will also work with -xs, -sm, -md, -lg

Vertical alignment

You can align columns verticaly inside the grid:

<div class="grid mod-top-xs"> ... <div class="grid mod-middle-xs"> ... <div class="grid mod-bottom-xs"> ... Tip: It will also work with -xs, -sm, -md, -lg
