Buttons

Buttons

<button class="button">...</button> <button class="button mod-outline">...</button> <button class="button mod-link">...</button> <button class="button mod-link mod-invert">...</button> Tip: Can be used as <a href="#">, for links.

Palette

<button class="button palette-primary">...</button> <button class="button palette-secondary">...</button> <button class="button palette-success">...</button> <button class="button palette-warning">...</button> <button class="button palette-error">...</button>

Size

<button class="button mod-small">...</button>

Counter

<button class="button mod-counter">Counting button<span class="button-counter">7</span></button>

Icons

States

<button class="button is-loading">...</button> <button class="button is-success">...</button> <button class="button is-error">...</button>

Disabled

<button class="button" disabled>...</button> Tip: Buttons can also be disabled with a class class="button is-disabled", but you will not be able to use a tooltip on it.

Group of buttons

You can wrap buttons to have a group of buttons.

<div class="button-group"> <button class="button">...</button> <button class="button">...</button> </div>

More button

You can use button's group to add more action to a button.

<div class="button-group mod-more"> <button class="button palette-primary">Main action</button> <button class="button palette-primary mod-more"></button> </div>

Example

Edit your buttons theme in components.override.scss:

$button: ( margin: 1rem, hover: ( box-shadow: 0 0 2px rgba(0, 0, 0, .15) ) ); $theme: _set($theme, "components.button", $button);
