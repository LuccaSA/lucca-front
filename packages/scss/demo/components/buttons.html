Buttons

Buttons

<button class="button">...</button> <button class="button mod-flat">...</button> <button class="button mod-wired">...</button> <button class="button mod-filling">...</button> Tip: Can be used as <a href="#">, for links.
Tip: You can use mod-link, to give a link aspect to your button.

Palette

<button class="button">...</button> <button class="button palette-primary">...</button> <button class="button palette-secondary">...</button> <button class="button palette-success">...</button> <button class="button palette-warning">...</button> <button class="button palette-error">...</button>

Size

<button class="button size-smaller">...</button> <button class="button size-small">...</button> <button class="button size-big">...</button> <button class="button size-bigger">...</button>

States

<button class="button is-loading">...</button> <button class="button is-success">...</button> <button class="button is-error">...</button>

Disabled

<button class="button" disabled>...</button> Tip: Buttons can also be disabled with a class class="button is-disabled", but you will not be able to use a tooltip on it.

Group of buttons

You can wrap buttons to have a group of buttons.

<div class="button-group"> <button class="button">...</button> <button class="button">...</button> </div>

It will also work with button's mods.

Radio buttons

You can use button group as radio form:

<div class="button-group"> <div class="button-group-item"> <input class="button-group-input" type="radio" name="radioButtonID" id="radioButton1" checked> <label class="button" for="radioButton1">Radio button</label> </div> <div class="button-group-item"> <input class="button-group-input" type="radio" name="radioButtonID" id="radioButton2"> <label class="button" for="radioButton2">Radio button</label> </div> </div>

Checkbox buttons

You can use button group as checkbox form:

<div class="button-group"> <div class="button-group-item"> <input class="button-group-input" type="checkbox" id="checkboxButton1" checked> <label class="button" for="checkboxButton1">Checkbox button</label> </div> <div class="button-group-item"> <input class="button-group-input" type="checkbox" id="checkboxButton2"> <label class="button" for="checkboxButton2">Checkbox button</label> </div> </div>
