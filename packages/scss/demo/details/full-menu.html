Implementation breakdown

Class Name Role States
<aside class="navSide">

The aside wraps the whole menu. It is responsible for:

  • fixing it to the left
  • Giving it its size (width and height)
  • Taking in account the offset due to external elements (such as a overhead menu)
  • Displaying the scrollbar when needed
  • Setting the mod-compact if desired
  • mod-compact displays a narrower menu without any submenu
<nav class="navSide-scrollWrapper">

A simple wrapper. It is responsible for:

  • fixing the menu width despite the possible presence of a scrollbar
<div class="navSide-item">

The container of a main menu item and its sub-menu. It is responsible of

  • Hidding/Showing the submenu
  • Indicating if the section is active
  • is-open to display the submenu
  • is-active to signify the section is active
<a class="navSide-item-link">

The item displayed in the bar. It can contain :

  • an icon <i>
  • a <span class="navSide-item-link-title">
If it has a submenu, it must ends with a <i class="lucca-icon navSide-item-arrow">arrow_east_thin</i>

  • is-active to show it is selected when it has no submenu
<nav class="navSide-item-subMenu">

The submenu wrapper. It is responsible for:

  • The sub-menu expand animation through max-height
<a class="navSide-item-subMenu-link">

The submenu item. It is a just simple link. When active, the parent <div class="navSide-item"> should be active to.

  • is-active to signify this link is the active section
