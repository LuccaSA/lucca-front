Lucca-icons Demo

Lucca-icons can be used in two ways:
  • using ligature: add the lucca-icon class to your element and use the corresponding ligature (written here below each icon) eg.: <i class="lucca-icon">user_roles</i>
  • using classes: add the lucca-icon class and use the ligature in dash-case as a class. eg.:<i class="lucca-icon user-roles"></i>
