# CalloutFeedbackList

Components to communicate information and states to the user.

**Storybook:** `Documentation/Feedback/Callout Feedback List/HTML&CSS`

### CSS Classes

| Class | Type |
|-------|------|
| `.calloutFeedbackList` | Base |
| `.calloutFeedbackList-item` | Base |
| `.calloutFeedbackList-item-description` | Base |
| `.mod-outlined` | Modifier |
| `.mod-ghost` | Modifier |
| `.mod-S` | Modifier |

### When to use

- Success/error messages
- Important alerts
- Contextual information

### When not to use

- Main content
- User actions

### Accessibility

- Use aria-live for dynamic messages
- Associate appropriate role (alert, status)
- Do not rely solely on color
