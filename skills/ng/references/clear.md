# Clear

Components for typography and textual content.

**Storybook:** `Documentation/Texts/Clear/Angular/Basic`

### Imports

```typescript
import { ClearComponent } from '@lucca-front/ng/clear';
```

### Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `size` | `'S' | null` | `null` | Which size should the clear be? Defaults to small |
| `palette` | `Palette` | `none` | Which palette should be used for the entire clear |

### Examples

```html
<lu-clear............>...</lu-clear>
```

### When to use

- Text formatting
- Labels
- Badges
- Tags

### When not to use

- Interactive actions
- Forms

### Accessibility

- Use logical heading hierarchy
- Ensure sufficient text contrast
- Avoid text in images
