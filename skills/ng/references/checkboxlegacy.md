# CheckboxLegacy

Components for user data input and validation.

**Storybook:** `Documentation/Forms/Checkbox Legacy/Basic`

### CSS Classes

| Class | Type |
|-------|------|
| `.checkboxesfield` | Base |
| `.checkboxesfield-label` | Base |
| `.checkbox-input` | Base |
| `.is-incomplete` | State |

### When to use

- Data entry
- Forms
- Configuration
- Filters

### When not to use

- Read-only data display
- Navigation

### Accessibility

- Associate each field with a label using for/id
- Provide explicit error messages
- Support keyboard navigation
- Indicate required fields
