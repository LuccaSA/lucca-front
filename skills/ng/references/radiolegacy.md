# RadioLegacy

Components for user data input and validation.

**Storybook:** `Documentation/Forms/Radio Legacy/Basic`

### CSS Classes

| Class | Type |
|-------|------|
| `.radiosfield` | Base |
| `.radiosfield-label` | Base |
| `.radio-input` | Base |

### When to use

- Data entry
- Forms
- Configuration
- Filters

### When not to use

- Read-only data display
- Navigation

### Accessibility

- Associate each field with a label using for/id
- Provide explicit error messages
- Support keyboard navigation
- Indicate required fields
