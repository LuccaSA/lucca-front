# Labels

Components for typography and textual content.

| Class | Type |
|-------|------|
| `.label` | Base |
| `.label-icon` | Base |

### When to use

- Text formatting
- Labels
- Badges
- Tags

### When not to use

- Interactive actions
- Forms

### Accessibility

- Use logical heading hierarchy
- Ensure sufficient text contrast
- Avoid text in images
