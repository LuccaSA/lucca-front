# ErrorPage

Components to communicate information and states to the user.

**Storybook:** `Documentation/Feedback/Error/Basic`

### CSS Classes

| Class | Type |
|-------|------|
| `.errorPage` | Base |
| `.errorPage-section` | Base |
| `.errorPage-section-info` | Base |

### When to use

- Success/error messages
- Important alerts
- Contextual information

### When not to use

- Main content
- User actions

### Accessibility

- Use aria-live for dynamic messages
- Associate appropriate role (alert, status)
- Do not rely solely on color
