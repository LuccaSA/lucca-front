# List

Components to display lists and data collections.

**Storybook:** `Documentation/Listings/List/Basic`

### CSS Classes

| Class | Type |
|-------|------|
| `.list` | Base |
| `.list-item` | Base |
| `.list-item-content` | Base |
| `.mod-onlyIcon` | Modifier |
| `.mod-S` | Modifier |
| `.mod-draggable` | Modifier |

### When to use

- Collection display
- Data tables
- Item lists

### When not to use

- Single element
- Forms

### Accessibility

- Use semantic structures (table, ul, ol)
- Provide headers for tables
- Support accessible sorting and pagination
