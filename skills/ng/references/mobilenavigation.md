# MobileNavigation

Components for application navigation.

**Storybook:** `Documentation/Navigation/MobileNavigation/Basic`

### CSS Classes

| Class | Type |
|-------|------|
| `.mobileNavigation` | Base |
| `.mobileNavigation-list` | Base |
| `.mobileNavigation-list-item` | Base |
| `.mod-XS` | Modifier |

### When to use

- Page navigation
- Menus
- Breadcrumbs
- Pagination

### When not to use

- Actions (use Button)
- Data display

### Accessibility

- Use appropriate nav landmarks
- Indicate current page with aria-current
- Support keyboard navigation
