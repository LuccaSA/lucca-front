# FileDropzone

Components for user data input and validation.

**Storybook:** `Documentation/File/FileDropzone/Angular/Basic`

### Imports

```typescript
import { FileDropzoneComponent } from '@lucca-front/ng/file-upload';
```


### When to use

- Data entry
- Forms
- Configuration
- Filters

### When not to use

- Read-only data display
- Navigation

### Accessibility

- Associate each field with a label using for/id
- Provide explicit error messages
- Support keyboard navigation
- Indicate required fields
