# MenuSecondary

Components for application navigation.

**Storybook:** `Documentation/Navigation/Menu Secondary/Basic`

### CSS Classes

| Class | Type |
|-------|------|
| `.navSide` | Base |
| `.navSide-wrapper` | Base |
| `.navSide-mainSection` | Base |
| `.mod-mobileToggle` | Modifier |
| `.is-active` | State |

### When to use

- Page navigation
- Menus
- Breadcrumbs
- Pagination

### When not to use

- Actions (use Button)
- Data display

### Accessibility

- Use appropriate nav landmarks
- Indicate current page with aria-current
- Support keyboard navigation
