# Icon

Core Prisme component shared across Lucca Front.

## Import

```typescript
import { IconComponent } from '@lucca/prisme/icon';
```

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `alt` | `string` | Information conveyed by the screen reader |
| `size` | `'XXS' | 'XS' | 'S' | 'M' | 'L' | 'XL' | 'XXL'` | Which size should the icon be? XXS to XXL |
| `color` | `'primary' | 'secondary' | 'product' | 'error' | 'warning' | 'success' | 'light' | 'placeholder' | 'inherit'` | Changes the color of the icon (inherit by default) |
