# Button

Core Prisme component shared across Lucca Front.

## Import

```typescript
import { ButtonComponent } from '@lucca/prisme/button';
```

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `size` | `'M' | 'S' | 'XS'` | Change the size of the Button |
| `palette` | `Palette` | Applies a color palette to the Button |
| `state` | `'default' | 'loading' | 'error' | 'success'` | Modifies the state of the Button |
