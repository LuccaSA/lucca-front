Checkboxes

<label class="checkbox"> <input class="checkbox-input" type="checkbox" /> <span class="checkbox-label">Checkbox</span> </label> Tip: To disable a checkbox, add a disabled attribute to the checkbox input.

Checkboxes mod-small

<div class="checkbox mod-small"> <input class="checkbox-input" type="checkbox" id="checkboxID"> <label class="checkbox-label" for="checkboxID">Checkbox</label> </div> Tip: To disable a checkbox, add a disabled attribute to the checkbox input.

Inline

<label class="checkbox mod-inline"> <input class="checkbox-input" type="checkbox" /> <span class="checkbox-label">Checkbox</span> </label>

Incomplete checkbox

For an incomplete checkbox, add class is-incomplete to input checkbox-input.
