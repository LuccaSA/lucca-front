# multi-select — Content

## Contenu et rédaction

* Le texte du label répond de manière claire et concise à la question de l’utilisateur “qu’est-ce que je sélectionne ?”.
* Le label reste descriptif, minimaliste : évitons donc le ton “parlé” ici.
* Le message d’aide doit tenir sur 2 lignes grand maximum. 1 seule dans l’idéal. N’oublions pas le point final au message d’aide.
* Tous les intitulés (label, placeholder, liste des valeurs, message d’aide) doivent comprendre une capitale au premier mot. Le reste demeurant en minuscule.
