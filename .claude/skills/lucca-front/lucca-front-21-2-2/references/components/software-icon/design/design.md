## Cas d'usage

Les Software icons doivent être utilisées uniquement pour représenter un logiciel Lucca :

1. Dans le catalogue de logiciels.
2. Pour naviguer vers un des logiciels (depuis la page d’accueil ou le menu de logiciels).
3. Pour représenter un logiciel au sein d'un autre logiciel Lucca (par exemple, dans l’Assistant Paie).

### Wrapper

Les Software icons peuvent être affichées sous forme de wrapper, par exemple dans les tableaux, pour afficher une liste de logiciels. Un nombre maximal peut être défini, les logiciels masqués restent accessibles via un Popover.

Un Tooltip s'affiche au survol des icônes pour indiquer le nom du logiciel.
