Containers

To wrap your content in a responsive container, use <div class="container">...</div>. Check templates section to see live examples.

You can reduce or increase the container width by adding a mod: mod-XXXS, mod-XXS, mod-XS, mod-S, mod-M, mod-L, mod-XL, mod-XXL or mod-XXXL.

If you have a sticky header or filter, you can add an extra space using this mod: mod-withStickyHeader.
