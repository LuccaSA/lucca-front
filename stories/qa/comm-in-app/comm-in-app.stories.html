Toasts Comm In App Work in progress

Toasts Comm In App allow you to show information about new module, on a corner (bottom left) of your app.

warningWarning: You have to manually create and kill your toasts for now, they will be added in the NG package in the future.

You have to use the same code as toast, BUT you need to add the mod-commInApp class to the div.toasts

If you want to use the same template for your comm In app, we recommand you to copy this code into your innerhtml toast item

Banner Comm In App Work in progress

Banner Comm In App allow you to show information about new module, on a banner at the top of your app.

The module has a new look, a new display is available.

To display this banner with a light mod, just add the class "mod-light" on the div "communicationBanner"
