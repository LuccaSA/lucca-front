# Scroll box

## Description

Le composant **Scroll box** fait partie de la catégorie **Structure** du design system Lucca Front.

Composants pour structurer la mise en page.

**Story path:** `Documentation/Structure/ScrollBox`


## Imports

## Utilisation

- Organisation du contenu
